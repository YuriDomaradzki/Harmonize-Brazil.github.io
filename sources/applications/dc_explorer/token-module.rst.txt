Create Token
============

.. image:: ../../img/dc-explorer/token.png
    :alt: Token Modal
    :align: center

Token Features
+++++++++++++++

-   You can create a personal token in **Harmonize Brazil Authentication** https://brazildatacube.dpi.inpe.br/auth-app/auth/login;
-   In the center of the page click on **Login**, after that you will be redirected to the login page;
-   Inside the window, we display the authentication page. You can either **create an account** or **login** the system. After that, you will be redirected to the user page of Harmonize Brazil Authentication;
-   You can access the **Personal Access Token** page by clicking on the menu;
-   At the top of the page, you can create a token by clicking in **Add New Personal Token**;


Additional Features
++++++++++++++++++++

-   You can copy token by clicking **copy icon** on the left of the token;
-   You can also view token by clicking **visualization icon** on the left of the copy icon;
