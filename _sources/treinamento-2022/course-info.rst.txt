Informações sobre o curso
-------------------------

O treinamento 2022 apresenta os conceitos básicos de séries temporais e cubos de dados, bem como a forma de utiliza-los, através da linguagem Python e R, juntamente aos serviços STAC, WTSS, WLTS e pacote SITS, tecnologias e dados produzidos pelo time do projeto `Brazil Data Cube <http://brazildatacube.dpi.inpe.br/>`_.


:ref:`course-treinamento-2022-day1`

- Abertura
- Visão geral do projeto BDC
- Geração das coleções e cubos de dados
- Criação conta BDC e token  e Preparação do ambiente Kaggle
- Plataforma BDC Explorer
- Uso do serviço STAC em Python

:ref:`course-treinamento-2022-day2`

- TerraCollect
- Uso do serviços WTSS em Python
- Uso do serviços Sample em Python
- Uso do serviço WLTS em Python

:ref:`course-treinamento-2022-day3`

- Uso do serviço STAC em R
- Uso do serviço WLTS em R
- Uso do serviço WTSS em R
- Introdução ao SITS R package

- SITS R package
