Python Client Library
=====================


If you want to access trajectory from Python programs, refer to: `Python Client Library for WLTS <https://github.com/brazil-data-cube/wlts.py>`_.


R Client Library
================


If you want to access trajectory from R programs, refer to: `rwlts <https://github.com/brazil-data-cube/rwlts>`_.


WLTS OpenAPI v3 Specification
=============================


For a detailed explanation of the WLTS service and its formal specification, please refer to: `WLTS OpenAPI 3 Specification <https://github.com/brazil-data-cube/wlts-spec>`_.


WLTS Server
===========


For a server implementation, refer to: `WLTS Server <https://github.com/brazil-data-cube/wlts>`_.
