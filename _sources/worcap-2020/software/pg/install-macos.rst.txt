Instalação do PostgreSQL e PostGIS para macOS
*********************************************


A maneira mais simples de realizar a instalação do PostgreSQL e PostGIS no macOS é através da distribuição `Postgres.app <https://postgresapp.com/>`_. Faça o download da versão ``PostgreSQL 11.6 / PostGIS 2.5.3`` e arraste o arquivo baixado para sua pasta de aplicativos. Isso será o suficiente para instalar o PostgreSQL e o PostGIS.
