Instalação do QGIS para macOS
*****************************


Na página principal do aplicativo `QGIS <https://qgis.org/en/site/>`_, selecione a opção ``Download Now``, como mostrado na figura abaixo.


.. image:: ../../../img/worcap-2020/software/qgis/install/homepage-qgis.png
    :alt: Homepage QGIS
    :width: 800
    :align: center


Na página de `Downloads <https://qgis.org/en/site/forusers/download.html>`_, selecione a opção ``Download for macOS`` e baixe a versão ``QGIS macOS Installer Version 3.4``, conforme mostrado na figura abaixo.


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-download-macos-installer.png
    :alt: QGIS macOS Installer Version 3.4
    :width: 800
    :align: center


Após concluir o download do instalador, siga os passos mostrados a seguir::


**1.** Abra o instalador. Uma janela como a mostrada abaixo será aberta. O QGIS é baseado em diversos softwares livres. Nessa janela você deverá aceitar os termos da licença do QGIS e de outros sistemas ou bibliotecas. Pressione ``Agree`` para continuar:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-installer-01.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**2.** Em seguida, será realizada uma verificação do pacote de instalação:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-installer-02.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**3.** Após a verificação, será aberta uma nova janela que possibilitará que você arraste o símbolo do QGIS para a pasta de aplicações (``Applications``).


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-installer-03.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**4.** O conteúdo do pacote será copiado para a pasta de aplicações:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-installer-05.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**5.** Durante alguns minutos, o ícone do ``QGIS`` na janela de aplicativos é apresentado desabilitado, como mostrado na figura abaixo:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-installer-04.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**6.** Ao final da instalação o ícone do QGIS estará habilitado na pasta de aplicativos, como mostrado abaixo:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-installer-06.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**7.** Através do ícone do QGIS na janela acima ou do *Launchpad* (janela abaixo), execute o aplicativo QGIS Desktop:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-launchpad.png
    :alt: Lançando o QGIS Desktop
    :width: 800
    :align: center


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-launchpad-icon.png
    :alt: Lançando o QGIS Desktop
    :width: 800
    :align: center


**8.** Em alguns casos, o ``macOS`` pode apresentar uma janela de aviso indicando que o QGIS não pode ser aberto. Neste caso, abra a janela de preferências do ``macOS`` e selecione a opção segurança. Pressione a opção 'Abrir mesmo assim' para permitir a inicialização do QGIS:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-not-open.png
    :alt: Tela Principal QGIS Desktop
    :width: 800
    :align: center


.. image:: ../../../img/worcap-2020/software/qgis/install/macos-preferences.png
    :alt: Tela Principal QGIS Desktop
    :width: 800
    :align: center


.. image:: ../../../img/worcap-2020/software/qgis/install/macos-preferences-security.png
    :alt: Tela Principal QGIS Desktop
    :width: 800
    :align: center


**9.** Se a instalação foi concluída com sucesso, ao iniciar o aplicativo ``QGIS``, uma janela como a mostrada abaixo será exibida:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-macos-welcome-2.png
    :alt: Tela Principal QGIS Desktop
    :width: 800
    :align: center


.. note::

    Mais informações sobre a instalação no ``macOS`` podem ser encontradas em: `QGIS Installers <https://www.qgis.org/en/site/forusers/alldownloads.html#mac-os-x-macos>`_.
