Instalação do QGIS para Microsoft Windows 10
********************************************


Na página principal do aplicativo `QGIS <https://qgis.org/en/site/>`_, selecione a opção ``Download Now``, como mostrado na figura abaixo.


.. image:: ../../../img/worcap-2020/software/qgis/install/homepage-qgis.png
    :alt: Homepage QGIS
    :width: 800
    :align: center


Na página de `Downloads <https://qgis.org/en/site/forusers/download.html>`_, selecione a opção ``Download for Windows`` e baixe a versão ``QGIS Standalone Installer Version 3.4 (64 bit)``, conforme mostrado na figura abaixo.


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-download-windows-standalone-installer.png
    :alt: QGIS Standalone Installer Version 3.4 - (64 bit)
    :width: 800
    :align: center


Após concluir o download do instalador, siga os passos mostrados a seguir:


**1.** Abra o instalador:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-standalone-installer-01.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**2.** Uma janela com o logo do QGIS será aberta. pressione ``Próximo`` para continuar:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-standalone-installer-02.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**3.** O QGIS é baseado em diversos softwares livres. Nessa janela você deverá aceitar os termos da licença do QGIS e de outros sistemas ou bibliotecas. Pressione ``Eu Concordo`` para continuar:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-standalone-installer-03.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**4.** Se você possui um disco SSD de pequeno porte, sugerimos alterar o local da pasta de destino do QGIS para um disco com maior capacidade, pois a instalação irá ocupar em torno de ``1.93GB``. Aponte para a pasta desejada e pressione ``Próximo`` para continuar:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-standalone-installer-04.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**5.** Nessa janela você não precisará baixar os dados de exemplo, pois preparamos diversos conjuntos de dados para o curso que serão compartilhados com todos. Pressione ``Instalar`` para continuar:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-standalone-installer-05.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**6.** Essa janela mostra apenas que o processo de instalação encontra-se em progresso:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-standalone-installer-06.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**7.** Ao final da instalação, uma janela como mostrada abaixo será apresentada. Pressione ``Terminar`` para finalilzar a instalação:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-standalone-installer-07.png
    :alt: QGIS Standalone Installer
    :width: 800
    :align: center


**8.** No menu ``Iniciar``, localize o grupo de aplicativos ``QGIS 3.4``. Iremos utilizar o ``QGIS Desktop 3.4.14``:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-windows-desktop.png
    :alt: Lançando o QGIS Desktop
    :width: 800
    :align: center


**9.** Se a instalação foi concluída com sucesso, ao iniciar o aplicativo ``QGIS Desktop 3.4.14``, uma janela como a mostrada abaixo será exibida:


.. image:: ../../../img/worcap-2020/software/qgis/install/qgis-tela-principal-windows.png
    :alt: Tela Principal QGIS Desktop
    :width: 800
    :align: center


.. note::

    Mais informações sobre a instalação no ``Microsoft Windows 10`` podem ser encontradas em: `QGIS Installers <https://www.qgis.org/en/site/forusers/alldownloads.html#windows>`_.
